# TheBlogger--TwitterBootstrap
TheBlogger is created with HTML5, CSS3, javaScript and jQuery. This Project is created for the client in Houston, Texas. It is a blogging
website, that has different categories.

<ul>
 <li>

 </li>
  <li>

 </li>
 <li>

 </li>
 <li>

 </li>
 <li>

 </li>

 </li>
